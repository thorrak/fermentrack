

Getting Started with Fermentrack
====================================

You can get started with Fermentrack in 3 steps:

1. Prepare the Raspberry Pi
2. Install Fermentrack
3. Configure Fermentrack

Additionally, once Fermentrack is installed, optional support for other applications can be installed as well.
